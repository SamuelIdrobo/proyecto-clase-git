# *Proyecto git*

inicio del proyecto

Jhon Jairo Ramirez 0907

El proyecto consiste en el diseño y construcción de un sistema web integral de comercio electrónico, orientado a optimizar la experiencia de compra de los usuarios y facilitar la administración del negocio. La plataforma abarca todo el ciclo de venta digital: desde el registro seguro de clientes y la exploración de un catálogo de productos con paginación, hasta la gestión dinámica del carrito de compras y un proceso de checkout optimizado. Paralelamente, integra un módulo administrativo exclusivo para la gestión del inventario, control de pedidos y generación de reportes de ventas detallados.
